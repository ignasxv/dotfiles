# User-Level Claude Instructions

These instructions apply to all projects.

## git
- When generating PRs unless needed don't add yourself as the author and don't add "generated by claude" marking in your PRs
- For github commits or PRs always use gh CLI and not any other MCP

## General Preferences
- Be concise and direct in responses
- Prefer editing existing files over creating new ones
- Avoid over-engineering — only make changes that are necessary
- Do not add comments, docstrings, or type annotations to code you didn't change

When logging lessons or knowledge, use the daily-knowledge database.
